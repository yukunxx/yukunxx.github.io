Redis

  • Redis是远程字典服务器
  • 由C语言编写，Key-Value数据库，基于内存
  • 分布式缓存、内存存储以及持久化、高可用架构、（缓存穿透、击穿、雪崩）、分布式锁、队列….

key操作命令

  • keys *：查找所有的key
  • type key ：查看key的类型
  • del key：删除key（阻塞）
  • unlink key：删除key(非阻塞)
  • ttl key：查看key的过期时间 （-1：永不过期 -2：已过期）
  • expire key 秒：设置key的过期时间
  • move key dbindex[0-15]：移动key到指定数据库
  • select dbindex：切换数据库 [0-15] 默认16个数据库
  • dbsize：查看当前数据库key的数量
  • flushdb：清空当前库
  • flushall：清除全部库

数据类型（10种）

String

基本语法：set key value

（1）参数：

  • nx：没有则添加

  • xx：有则添加

  • get：弹出旧值
  • ex：过期时间（秒）
  • px：过期时间（毫秒）
  • exat：过期时间（UNIX时间戳）（秒）
  • pxat：过期时间（UNIX时间戳）（毫秒）
  • keepttl：保存过期时间

显然k1的过期时间没有被保存，直接失效了
